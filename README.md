# HorizonFix

An SKSE plugin that makes Skyrim's horizon actually make some sense

[Link to Nexus Mods](https://www.nexusmods.com/skyrimspecialedition/mods/184607)

## Contributing

Contributors are welcome. You can build the project by running the `buildRelease.ps1` powershell script from a visual studio dev powershell prompt

AI usage is allowed and was used as a tool for this project (for reviews, comments, and some RE), however, all code merged must be approved manually and understood/verified fully by its author and this project's maintainers.
